* Design adaptado para dispositivos móveis

## 💻 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/IdealBruno/Landing-Page.git
```

Entre na pasta:

```bash
cd Landing-Page
```

Abra o arquivo `index.html` no navegador.
